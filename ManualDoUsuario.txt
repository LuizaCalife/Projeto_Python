| O nosso projeto consiste em um sistema de rastreamento de despesas pessoais.                |
| No qual o usuário terá maior controle dos gastos e facilite sua visão diante                |
| suas contas. Contamos com diversas funcionalidades como: adição, leitura, atualização       |
| e deleção de transações manualmente, possuimos um menu intuitivo e completo podendo filtrar |
| seus gastos por categorias.                                                                 |

---O usuário começa sua navegação pelo menu podendo escolher entre diversas opções 
de transação, após a escolha ele será direcionado para a função escolhida e poderá realiza-lá.---

Em nosso menu contamos com várias opções das quais o usuário pode utilizar como por exemplo:
 1. Adicionar transação
 2. Listar transações
 3. Atualizar transação
 4. Deletar transação
 5. Filtrar transações por categoria
 6. Extrato por categoria

| Pensando em uma melhor experiência de usuário utilizamos um sistema de "Banco de Dados",       |
| no qual mesmo que o programa atual seja finalizado os dados colocados pelo usuário continuarão |
| lá na próxima execução.                                                                        |
